# Indian Heritage Hub (FSAD-PS05)

A React web app to explore Indian culture, heritage, historical places, and famous monuments.
Includes role-based dashboards: Admin, Cultural Enthusiast, Content Creator, Tour Guide.

## Run locally
```bash
npm install
npm run dev
```
